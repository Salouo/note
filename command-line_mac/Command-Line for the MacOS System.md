# Command-Line for the MacOS System

## 1. Enter a folder

```sh
cd your_path
```



## 2. Open a file which is text format

```sh
open -e example.txt
```



## 3. Go up levels in the directory structure



### 3.1 Go up one level

```sh
cd ..
```



### 3.2 Go up 2 levels

```sh
cd ../../
```



## 4. Go back to the base

```sh
cd ~
```
